📋 Getting Started

Task Master helps you organize and manage tasks from markdown files. You can load tasks from .md files, view them in different formats, and export your data.

🔄 Loading Tasks

  • Load MD Files: Import tasks from markdown files with checkbox format
  • Import Master File: Load a previously exported JSON master file
  • Duplicate Detection: Tasks are automatically merged to prevent duplicates

✅ Supported Task Formats

The following checkbox formats are supported in markdown files:

  • - [ ] Pending task
  • - [x] Completed task or - [✓] Completed task
  • - [_] In progress task or - [-] In progress task

📊 Phase and Task Structure

For better organization, use this structure in your markdown files:

## Phase 1: Foundation
**Task 1.1: Project Setup**
- [x] Initialize React project
- [ ] Set up build configuration

**Task 1.2: Core Features**
- [ ] User authentication system
- [ ] Dashboard implementation

🔍 Features

  • Search: Find tasks by title, notes, or source file
  • Filters: Filter by status (pending, completed, in-progress) or category
  • Views: Switch between card view and table view
  • Sorting: Click column headers in table view to sort
  • Export Filtered View: Export current filtered tasks to markdown format
  • Statistics: Real-time stats showing task completion rates

💾 Data Management

  • Auto-save: Tasks are automatically saved to local storage
  • Export Master File: Save all tasks as JSON for backup
  • Categories: Tasks are automatically categorized (Feature, Bug, Testing, Documentation)

⚡ Keyboard Shortcuts

  • Search: Click the search box or start typing to filter tasks
  • Modal: Press Escape to close modals
